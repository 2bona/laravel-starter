# Laravel Starter
Laravel 5.6 based simple starter project.

* User Authentication
* Social Login
 * Facebook
 * Github
* Role-Permissions for users
* Backend Theme
 * Bootstrap 4, CoreUI
 * Fontawesome 5
* Frontend Theme
 * Bootstrap 4, Now UI Kit
 * Fontawesome 5
* Posts Module
 * wysiwyg editor
 * file browser
* Categories Module
* External Libraries
 * Bootstrap 4
 * Fontawesome 5
 * CoreUI
